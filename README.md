## 1. 项目概述

### 1.1 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 林间树屋 |
| 游戏类型 | 2D 横版社交装扮 |
| 目标平台 | 微信小游戏 |
| 目标用户 | OC 圈爱好者、装扮类休闲玩家 |

> 是一个可以搭建树屋、摆放OC角色、互访好友房间、和好友互动的休闲社交小游戏

### 1.2 核心特点

- 可用提供的素材捏出自己的角色或者OC
- 可自由搭建属于自己的树屋，或者在好友的同意下在好友树屋中住下
- 布置属于自己的住处
